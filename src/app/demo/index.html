Demonstration

These demonstration pages are an example how angular-form-gen can be implemented. All data is stored localy in the browser session (there's no server backend) and a small artificial delay has been builtin to simulate backend communication.

Below are some pre-created example forms.

  • Click on the New button to create a new form.
  • Click on the Edit button to edit the schema.
  • Click on the Data button or the name of the form to enter or show data entries of the form.
